# Efektio flutter app

## Requirements

 - `cargo install`
 - `cargo install cargo-make`
 - `cargo install cargo-ndk`
 - `cargo install cargo-lipo`
 - `flutter`

+ Android NDK / XCode

## Setup

`cargo make init`

## Update for SDK

`cargo make build`

## Running the App

F5 in VS Code or `flutter run` in `app`
